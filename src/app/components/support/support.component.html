GitHub

GitHub is a website for collaborating on software projects. Transparent Salaries has a page on GitHub. On this page, you can view issues, communicate on an existing one, or open a new one.

Unfortunately, to open an issue or communicate on an existing one, you need to create an account on GitHub.

Although it's called issues, feel free to use it for any communication regarding Transparent Salaries, like, for example, feature requests, questions, or any other feedback.

Private issues

There's currently no way to file an issue privately; anything you write in an issue is public to everyone.

The reason for that is that I don't know of a need for filing issues privately. If you happen to face such need, please tell me by opening a public issue and I'll try to create a way for filing issues privately.
